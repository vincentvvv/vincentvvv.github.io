After learning object oriented programming, and a bit more experience with Unity, I finally decided to revisit this project. This version is a lot more better than the previous version. However, there are a lot of bugs in the game that I haven't got the chance to fix. When I get more time, and some more motivation, I plan on completing the game one day and hopefully draw my own sprites/ find an artist to make them.
